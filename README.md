## Overview

The **SplitBot Feedback Hub** is a public space for customers, testers, and contributors to:

- **File bugs**
- **Request features**
- **Discuss ideas**

All issues are triaged by the SplitBot core team. Feel free to comment, vote, or open a discussion.

---

## Getting Started

1️⃣ Browse the **Issues** tab and filter by `bug`, `enhancement`, or `feedback`.
2️⃣ If you have a broader idea, start a **Discussion**.
3️⃣ Follow the provided templates to ensure we have all the necessary information.
